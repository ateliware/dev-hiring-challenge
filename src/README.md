# GithubMiner

# Primeiro

Certifique-se de ter instalado em sua máquina o docker e o docker-compose

# Segundo 

Execute o comando "docker compose up dev --build" na raiz do projeto para baixar as imagens e construir os containers

# Terceiro 

Visite localhost:4000 para visualizar a aplicação em andamento

# Adminer

Gerenciador de banco de dados, visite localhost:8080
